
# 💡 Controle de LEDs por Botões com ESP32-S2

## Descrição do Projeto

Este projeto consiste em um sistema simples de controle de LEDs utilizando a placa ESP32-S2 programada em MicroPython. O circuito permite a interação direta do usuário por meio de três botões, onde cada botão é responsável por acionar um LED de sua respectiva cor. O projeto foi desenvolvido e testado no simulador Wokwi, possibilitando a validação do funcionamento antes da implementação física.

## Elementos Utilizados

* 1 ESP32-S2
* 3 LEDs (Vermelho, Azul e Amarelo)
* 3 Botões Pushbutton
* 3 Resistores
* Protoboard e fios de conexão
* MicroPython
* Simulador Wokwi.

![imagem](https://github.com/Pereira06z/LED-PUSHBUTTON/blob/main/LED%2BPUSHBUTTON.png)

## Conclusão

O projeto demonstra de forma prática o controle de saídas digitais a partir de entradas digitais, permitindo compreender conceitos fundamentais de sistemas embarcados, programação em MicroPython e utilização dos pinos GPIO da ESP32-S2.
